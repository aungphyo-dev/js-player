###### 27. What's the output?

```javascript
for (let i = 1; i < 5; i++) {
  if (i === 3) continue;
  console.log(i);
}
```

- A: `1` `2`
- B: `1` `2` `3`
- C: `1` `2` `4`
- D: `1` `3` `4`

<details>
<summary>
<b>Answer</b>
</summary>

#### Answer: C

The `continue` statement skips an iteration if a certain condition returns `true`.
